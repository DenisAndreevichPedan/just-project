# Just Project

## Prerequisites

- Node.js >= 20
- pnpm 10.30.1 (`corepack enable` to use the version from package.json)

## Getting Started

```bash
pnpm install
pnpm dev
```


> **Note:** This project uses pnpm only. Do not use npm or yarn.

## Project Structure

- `apps/web` — Next.js web application
- `packages/shared` — shared utilities and types
